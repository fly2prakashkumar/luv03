# **App Name**: Luv 03

## Core Features:

- Product Catalog: Display a catalog of skincare products with detailed descriptions and images.
- Shopping Cart: Allow users to add products to a shopping cart and manage their selected items.
- Secure Checkout: Implement a secure checkout process for users to purchase products with credit card or other payment options.
- AI Product Recommendation Tool: Provide personalized product recommendations based on user's skin type and concerns using AI. The AI will use its reasoning abilities to decide which recommendations would be best to show the user based on publicly available skincare knowledge.
- User Accounts: Enable users to create accounts to save their addresses and purchase history.
- Order Tracking: Allow users to track the status of their orders from placement to delivery.

## Style Guidelines:

- Primary color: Pastel Magenta (#F4B4E4) to evoke a sense of calm and luxury.
- Background color: Off-white (#F8F8FF) for a clean and minimalist aesthetic.
- Accent color: Pale rose (#FFB6C1) for highlighting important elements and calls to action.
- Body and headline font: 'Alegreya', a serif font that is elegant, intellectual, and contemporary.
- Use clean, line-based icons in a neutral color to complement the overall aesthetic.
- Implement a clean and minimalist layout with plenty of white space to highlight the products.
- Use subtle animations, like fade-in effects and smooth transitions, to enhance the user experience.
